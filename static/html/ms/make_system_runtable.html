Move the {{run_table_name}} run table to the system run tables. System run tables are visible to all experiments (or experiments on the {{instrument}} instrument); this move will replace any system run table with the same name. This run table, {{run_table_name}}, will be deleted from this experiment. While it's admissible to have global and instrument specific tables with the same name, this can be confusing to end users.
